This project is in collaboration with Daimler Trucks to reduce axle misalignment during assembly. In the factory, axles are installed onto a flipped truck chassis, where small lateral or angular errors can lead to costly rework later in production. Our goal is to detect these misalignments within allowed tolerances so corrections can be made early on the line.
